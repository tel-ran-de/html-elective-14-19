Einige HTML Tags

< abbr >

Dieses Tag definiert eine Abkürzung

< address >

Dieses Tag definiert die Kontaktinformationen(z.B Email Adresse,eine physische Adresse, eine URL usw.).

< aside >

Dieses Tag wird verwendet, um Inhalte zu identifizieren, die sich auf den Hauptinhalt beziehen, aber nicht der Hauptinhalt der Website darstellen(z.B verwandte Links,Inhalte usw.).

< code >

Dieses Tag definiert ein Stück des Computercodes.

< caption >

Dieses Tag definiert die Tabellenüberschrift

< cite >

Dieses Tag verweist auf ein Werk, eine kreative Arbeit.

< fieldset >

Dieses Tag wird verwendet, um zusammengehörige Felder eines Formulares zu gruppieren.

< figcaption >

Dieses Tag definiert eine Bildunterschrift für das Foto.

< legend >

Dieses Tag definiert eine Beschrifting für ein < fieldset > Element.

< mark >

Dieses Tag wird verwendet, um zu definieren,welchen Text hervorgehoben oder markiert werden soll.

< output >

Dieses Tag wird verwendet,um das Ergebnis einer Berechnung mithilfe JavaScript zu zeigen.

< optgroup >

Dieses Tag wird verwendet, um verwandte Optionen in einer Dropdownliste zu gruppieren.

< option >

Dieses Tag definiert eine Option in einer Dropdownliste.

< picture >

Dieses Tag regelt die Darstellung von Bildern, um sie für unterschiedliche Endgeräte anzupassen.

< progress >

Dieses Tag wird in Verbindung mit JavaScript verwendet, um den Fortschritt einer Aufgabe anzuzeigen.

< section >

Dieses Tag definiert einen Abschnitt in einem Dokument.

< samp >

Dieses Tag wird verwendet, um eine beispielhafte oder zitierte Ausgabe eines Computerprogramms darzustellen.

< summary >

Dieses Tag wird für Zusammenfassungen für den Inhalt eines < details > Elements verwendet.

< tfoot >

Dieses Tag definiert footer für eine Tabelle.

< thead >

Dieses Tag definiert header für eine Tabelle.

< time >

Dieses Tag definiert eine bestimmte Zeit(Datum/Urzeit).

< track >

Dieses Tag dient der Anzeige von Untertiteln oder der Ausgabe von Buildunterschriften für Sehbehinderte.

< meter >

Dieses Tag definiert eine skalare Messung.

< nav >

Dieses Tag definiert eine Reihe von Navigationslinks.

< menu >

Dieses Tag definiert eine Gruppe von Befehlen, die ein Benutzer ausführen oder aktivieren kann.

< menuitem >

Dieses Element ist das Kindelement eines menu-Elements(veraltet).

< keygen >

Dieses Element zeichnet ein Generator aus, mit dessen Hilfe ein kryptografisches Schlüsselpaar erzeugt werden sollte(veraltet).

< main >

Dieses Tag gibt den Hauptinhalt eines Dokuments an.

< embed >

Dieses Element repräsentiert einen Integrationspunkt für eine externe (in der Regel nicht-HTML) Anwendung oder interaktive Inhalte.(Plag-in)
